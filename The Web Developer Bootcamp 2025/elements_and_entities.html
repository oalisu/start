HTML Elements and Entities

In HTML, elements are the building blocks of web pages. They are defined by tags, which are enclosed in angle brackets. For example, the <p> tag defines a paragraph element.

Common HTML Elements

  • <h1> to <h6>: Header elements
  • <p>: Paragraph element
  • <a>: Anchor (link) element
  • <img>: Image element
  • <ul> and <ol>: Unordered and ordered list elements
  • <li>: List item element

HTML Entities

HTML entities are special codes that represent characters that have a special meaning in HTML or are not easily typed on a keyboard. They start with an ampersand (&) and end with a semicolon (;).

Common HTML Entities

  • &amp;: Represents the ampersand (&) character
  • &lt;: Represents the less-than (<) character
  • &gt;: Represents the greater-than (>) character
  • &quot;: Represents the double quotation mark (") character
  • &apos;: Represents the single quotation mark (') character
  • &nbsp;: Represents a non-breaking space

Element sub and sup

The <sub> and <sup> elements are used to define subscript and superscript text, respectively. Subscript text appears slightly below the normal line of text, while superscript text appears slightly above it.

Example of subscript: H2O (water molecule)

Example of superscript: E = mc2 (Einstein's mass-energy equivalence formula)
